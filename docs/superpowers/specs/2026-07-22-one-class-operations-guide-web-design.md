# 壹号教室全年运营指导互动网页设计规格

## 1. 背景与目标

壹号教室位于上海市嘉定区嘉定新城，面向三年级以上学生提供小学语数英同步辅导、初中语数英培优或薄弱科提升、作业托管、晚托、寒暑假班、一对一和一对多服务。机构已经正常营业，有稳定在读学生，但整体生源仍需增长，同时需要提高续费、转介绍和本地品牌影响力。

本次交付是一份用于向机构负责人转述运营方案的文字型互动网页。它不是运营后台，不连接微信、视频号、大众点评、美团或任何真实业务数据，也不执行自动发布。

网页目标：

- 完整呈现双方已经确认的全年运营方法和执行建议。
- 让负责人在手机或电脑上快速理解获客、续费和转介绍的闭环。
- 把全年计划、日常动作、平台分工、AI自动化、学习路线和90天计划组织成可执行手册。
- 支持直接发送一个 HTML 文件、离线打开和打印为 PDF。

## 2. 目标读者与使用场景

主要读者是壹号教室负责人、兼职运营人员和授课老师。读者可能不了解内容运营或自动化，因此页面使用中文和普通业务语言，不依赖技术背景。

典型场景：

- 兼职运营人员向负责人完整讲解方案。
- 负责人在手机上分章节阅读并确认人员分工。
- 老师查看素材提交要求和授权规则。
- 团队打印或导出 PDF，作为内部讨论材料。

## 3. 交付形式

- 单个自包含 HTML 文件，文件名为 `one-class-operations-guide.html`。
- HTML 内包含全部 CSS 和 JavaScript，不依赖构建工具或服务器。
- 文字版，不使用门店照片、学生照片或通用图库图片。
- 外部教程使用普通链接；离线阅读不受影响，点击教程链接时需要联网。
- 页面在 Chrome、Edge 和主流手机浏览器中可用。

## 4. 信息架构

页面按以下顺序组织：

1. 执行摘要：机构现状、运营目标、推荐路线和前三个月重点。
2. 三条增长路径：新客获客、老生续费、口碑与转介绍。
3. 各平台定位：朋友圈、视频号、大众点评、美团和微信私聊的职责。
4. 全年宣传计划：12个月主题、重点内容和转化目标，以及四个季度的任务。
5. 每月、每周、每天动作：兼职运营的6至8小时周节奏。
6. 老师素材提交：固定提交格式、素材状态和授权检查。
7. 平台操作指南：各平台发布内容、维护重点和统一行动入口。
8. 好评外传：评价邀请、授权、脱敏和一条评价复用为多份内容的流程。
9. AI工具组合：飞书多维表格、普通AI工具、剪映、扣子、Dify和n8n的分层使用。
10. 自动化边界：适合自动化、必须人工处理和明确禁止的动作。
11. 自用工具路线：运行8至12周后再决定是否开发，以及功能优先级。
12. 官方学习入口：飞书、Dify、n8n和扣子的官方资料。
13. 8周学习计划：每周主题与必须产出的成果。
14. 30/60/90天执行计划：基础建设、优化转化和验证增长。
15. 人员分工与预算：兼职运营、负责人、老师和门店人员的职责。
16. KPI与月度复盘：访问、咨询、到店、报名、续费和转介绍。
17. 合规与隐私：未成年人信息、真实评价、宣传用语、资质和平台规则。
18. 首月任务清单：可在浏览器中勾选并保存进度。

## 5. 交互设计

- 桌面端使用左侧固定章节导航，移动端使用顶部可展开目录。
- 点击目录平滑滚动到对应章节，并高亮当前章节。
- 长章节可展开或收起；默认展示执行摘要和当前行动，避免首次打开信息过载。
- 提供“全部展开/全部收起”和“打印/导出 PDF”命令。
- 首月任务清单支持勾选，状态写入当前浏览器的 `localStorage`。
- 页面顶部显示清单完成比例。该比例只代表本地任务勾选，不代表经营业绩。
- 外部教程链接在新标签页打开，并标记为官方资料。
- JavaScript 不可用时，所有核心文字仍然完整可读。

## 6. 视觉与响应式规则

- 风格为安静、专业、面向本地教育机构的工作手册，不制作营销式首页。
- 使用白色、浅灰作为主体背景，深绿色用于关键路径，蓝色用于工具与信息，黄色用于提醒，红色仅用于风险警告。
- 首页第一屏直接呈现标题、机构定位、推荐方案和90天目标摘要。
- 章节采用全宽内容区域和细分隔线，不使用层层嵌套的卡片。
- 表格在窄屏下转换为逐项信息块或允许横向滚动，文字不得溢出。
- 正文字号、行高和内容宽度适合长文阅读；不随视口宽度连续缩放字体。
- 打印样式移除导航和交互按钮，展开全部内容，保留清晰分页与表格标题。

## 7. 内容原则

- 保留已确认的真实业务范围和嘉定新城本地定位。
- 明确推荐“AI辅助的轻量运营系统”，而不是立即开发大型平台。
- 不把每天维护误写成每天在大众点评和美团强行发布。
- 不承诺固定招生人数、提分、排名或投资回报。
- 三个月有效咨询增长20%至40%只作为调整后的目标区间，不作为结果保证。
- 正确说明当前无宣传预算，首月建议0至300元，后续工具预算300至800元；付费推广须在资质、承接和追踪条件齐备后再小范围测试。
- 强调白天咨询由负责人或门店老师承接，兼职运营负责晚间内容、系统和复盘。

## 8. 隐私、合规与安全

- 不在页面中嵌入任何真实学生姓名、学校、成绩单编号、联系方式或未授权影像。
- 家长评价只能真实邀请，不购买、代写或通过奖励换取指定好评。
- 评价和学生案例公开前必须获得授权并完成脱敏。
- 文化课培训的课程、时段、收费和宣传需符合机构实际资质、平台类目及上海当地监管要求。
- 自动化只用于收集、整理、提醒、生成初稿和汇总数据；不制作刷评、养号、爬取家长信息或模拟人工批量发布功能。
- 页面不收集、不上传、不向外部服务发送用户数据。

## 9. 验收标准

- 一个 HTML 文件即可打开，中文无乱码。
- 包含双方已确认的全部运营指导，不加入不存在的机构事实。
- 360像素宽手机、平板和桌面均无文字或控件重叠。
- 导航、章节折叠、全部展开/收起、打印和任务勾选可用。
- 刷新后任务勾选状态仍保留。
- 打印预览包含所有正文，不出现被折叠而缺失的章节。
- 外部官方教程链接准确且具有清楚标签。
- 页面在禁用 JavaScript 时仍可阅读完整核心方案。

## 10. 明确不做

- 不开发真实运营后台、账号登录、云端数据库或多人协作。
- 不接入微信、视频号、大众点评、美团 API。
- 不自动发布内容，不读取账号或家长数据。
- 不制作门店宣传照片、学生案例图片或视频。
- 不在本次网页中实现数据看板的真实数据录入和统计。
