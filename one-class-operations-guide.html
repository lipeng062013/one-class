壹号教室全年运营与宣传执行方案

这份手册回答一个具体问题：在没有固定宣传预算、由兼职人员晚间执行的条件下，如何持续做好朋友圈、视频号、大众点评、美团和口碑传播，并逐步建立一套AI辅助的运营系统。

三年级以上文化课与托管 每周运营投入约6-8小时 先自然增长，后小额测试 先跑流程，再开发工具
90天目标（不承诺） 稳定周更与线索追踪流程，有效咨询量较首月基线争取提升20%-40%，再决定是否小额投放或开发工具。
先看90天执行计划 查看全年宣传日历
01

执行摘要

+

壹号教室已经正常营业，有稳定在读学生和真实素材，但整体生源尚未达到理想规模。运营采用“双引擎”：用老生学习过程、续费和转介绍建立信任，同时通过本地搜索与短内容获得新咨询。

主营服务

三至六年级语数英同步辅导；初中语数英培优或薄弱科提升；作业托管、晚托、寒暑假班；一对一和一对多。

真实优势

小班制、固定老师、全程陪学；学习习惯、错题订正和阶段测评；接送、安全管理与离校流程。

推荐路线

采用AI辅助的轻量运营系统。共享素材库、固定模板、批量生成初稿，发布与家长沟通仍由人工完成。

为什么不立即开发平台

路线优势限制结论
纯手工轻运营零成本、马上开始容易断更，素材和数据分散仅适合短期过渡
AI辅助标准化运营低成本、可持续、每周可批量处理需要建立素材和审核习惯当前采用
自研运营平台长期效率和数据统一性更高过早开发容易做错需求运行8-12周后再评估
核心判断：内容不是为了把账号发得热闹，而是持续回答家长的四个问题：适合谁、能解决什么、服务过程是否可靠、下一步怎样咨询。
02

三条增长路径

+

新客获客

本地家长在朋友圈、视频号或点评/美团看到真实内容，进入门店页，咨询并预约到店测评或试听，最后报名。

老生续费

把课堂、作业、错题和阶段测评变成家长能看懂的过程证据，推动续费或升级服务。

口碑转介绍

在进步、满意和续费节点邀请真实评价，经授权和脱敏后用于口碑外传与老带新。

新客路径

本地曝光内容或搜索
查看证据环境、老师、案例
有效咨询年级、科目、需求
到店沟通测评或试听
报名跟进课程与服务匹配

续费路径

阶段反馈测评与建议
家长沟通看见过程
续费升级匹配下一阶段

03

各平台定位

+

各平台不能平均用力。每个平台承担不同任务，但最终都指向一个明确行动入口，例如“预约一次学习情况沟通”或“预约到店了解晚托流程”。

平台核心任务主要内容建议频率
朋友圈熟人信任、咨询转化教学日常、学习变化、家长反馈、班级名额每周4-5条
视频号本地曝光、专业表达课堂片段、老师讲题、学习方法、真实场景每周1-2条
大众点评承接主动本地搜索门店资料、环境、师资、课程特色、真实评价每天检查，每周维护
美团产品化和到店转化测评、体验、托管与阶段课程入口每天检查，每周维护
微信私聊/电话咨询、预约和成交需求判断、预约试听、回访与报名跟进有线索及时处理
运营分工：大众点评和美团承接已经在寻找机构的家长；视频号扩大本地触达；朋友圈持续建立信任；微信私聊和电话完成咨询与到店预约。

05

每月、每周、每天怎么做

+

执行原则是“每天检查、每周批量生产、按平台分发”。大众点评和美团不需要每天强行发布新内容。

每天晚上20-30分钟

  • 查看微信、大众点评、美团的新咨询、评论和评价。
  • 登记家长称呼、年级、科目、需求、来源和下次跟进时间。
  • 回复评价和私信，目标是不超过24小时。
  • 从老师当日素材中选一条可用内容，必要时发布朋友圈。
  • 记录访问、咨询、到店和报名变化。

06

老师怎样提交素材

+

老师不负责写宣传稿，只需要每周提交2组真实、可解释的素材。运营人员负责筛选、改写、脱敏和排期。

日期：
年级/科目：
今天解决了什么问题：
孩子有哪些变化：
可公开的照片或视频：
是否已获得家长公开授权：
补充说明：

素材状态

  • 待整理：老师刚提交，尚未判断用途。
  • 可发布：内容真实、信息完整、授权明确。
  • 已发布：记录平台和发布日期，避免短期重复。
  • 需授权/不可公开：存在正脸、姓名、学校、成绩单编号或其他隐私信息。
底线：没有明确授权的学生影像和评价不能公开。可先保留为内部教学记录，不能因为内容“很好看”就跳过授权。
07

各平台具体怎么运营

+

朋友圈：建立信任

一条内容尽量包含“一个具体场景 + 一个家长痛点 + 老师怎么处理 + 下一步入口”。真实课堂、老师工作和孩子学习过程通常比复杂海报更有说服力。

五年级孩子做应用题总是读完就下笔，错的并不是计算，而是没有先找数量关系。今天老师带他把题目分成“已知、问题、关系”三步，再完成同类题订正。同步辅导不只是把作业做完，更重要的是把错误背后的习惯找出来。需要了解孩子目前学习情况的家长，可以预约到店沟通。

视频号：本地曝光与专业表达

每条建议20-60秒，前3秒直接提出问题。固定栏目比临时想选题更容易坚持。

  • “孩子作业写得慢，真的是不专心吗？”
  • “数学错题抄了很多，为什么下次还会错？”
  • “托管班只是看着孩子写完作业吗？”
  • “五年级开始，哪些学习习惯最容易拉开差距？”

栏目可固定为：老师讲一个学习问题、一道题背后的思考方法、课堂或晚托真实片段、一周学习变化、接送和离校流程。

大众点评：承接主动搜索

  • 名称、地址、电话和营业时间准确一致。
  • 门头、前台、教室、学习区、安全设施和接送场景清晰。
  • 老师介绍只写真实教龄、年级与科目，不写无法证明的头衔。
  • 课程写清适合年级、班型、服务内容和上课方式。
  • 自然使用“嘉定新城、小学同步辅导、初中薄弱科提升、晚托”等真实业务词。
  • 检查家长常问问题，所有评价尽量在24小时内个性化回复。

遇到差评时先确认事实，说明处理动作并邀请线下沟通，不与家长公开争辩，也不公开学生信息。

美团：产品化与到店转化

页面先设置少量、容易理解的入口：学情沟通或阶段测评、小学同步辅导体验、初中薄弱科体验、晚托体验、寒暑假阶段课程。

每个产品写清适合对象、包含内容、时长、预约方式、使用限制，以及是否需要携带作业或试卷。价格和促销必须真实，并在机构资质、平台类目和当地规定允许的范围内上线。

咨询与成交：统一入口

不要在同一条内容里同时要求家长点赞、加微信、打电话和买套餐。统一使用一个行动入口，例如“预约一次学习情况沟通”。咨询时至少记录年级、科目、主要问题、意向服务、来源和下一次跟进时间。

08

好评外传：一条评价变成五份内容

+
真实反馈满意节点自然提出
询问授权明确可用范围
隐私脱敏姓名、学校、头像
提炼故事问题、动作、变化
多端复用按平台改写

评价邀请话术

感谢您这段时间对老师工作的认可。如果您愿意，可以在大众点评上真实分享一下孩子的学习体验，给附近正在选择辅导机构的家长提供参考。按您的实际感受写就可以。

复用方式

  • 朋友圈：讲清老师如何解决具体问题。
  • 视频号：让老师解释对应的学习方法。
  • 点评动态：展示学习习惯管理或服务过程。
  • 咨询案例：向有相似问题的家长说明服务方式。
  • 月度复盘：记录该类内容带来的咨询变化。
例子：家长说“以前每天写作业都要催，现在回来会先自己列任务”，可以转化为任务清单训练案例，而不是只截一张“老师很好”的泛泛好评。
禁止：购买评价、代写评价、让家长照模板写指定内容、用礼品交换指定好评、刷量或删除真实问题。

10

哪些能自动化，哪些必须人工

+

适合自动化

  • 每周提醒老师交素材
  • 新素材自动建立任务
  • 生成多平台初稿
  • 未授权内容标记禁发
  • 发布与回访提醒
  • 月底汇总数量和线索

必须人工

  • 发布前事实与隐私检查
  • 平台最终发布
  • 私信、评价和咨询回复
  • 案例是否适合公开
  • 课程与价格最终确认
  • 家长关系和投诉处理

明确禁止

  • 刷评与虚假评价
  • 自动养号和模拟人工
  • 抓取家长个人信息
  • 未授权批量发布
  • 把未脱敏学生资料交给通用AI
  • 绕过平台接口和规则
平台边界：微信、视频号、大众点评和美团没有明确开放并授权的发布接口时，不把“一键跨平台自动发布”作为目标。自动化重点是整理、提醒、初稿和统计。
11

以后适合开发哪些自用工具

+

先用飞书流程运行8-12周，只记录反复出现的真实痛点。确认高频需求后，再开发轻量内部后台。

  1. 老师手机上传素材：提交场景、学习问题、文件和授权信息。
  2. AI多平台草稿：依据壹号教室课程、语气和禁用词生成初稿。
  3. 审核与排期：显示待授权、待审核、可发布、已发布状态。
  4. 咨询与回访：记录来源、需求、预约和下一步提醒。
  5. 招生数据看板：展示咨询、到店、报名、续费和转介绍。
  6. 机构知识库：课程、师资、常见问题、服务流程和宣传禁用词。
第一版不做：平台账号托管、自动发布、评价抓取和复杂多人权限。先解决素材、内容、线索和复盘四个核心问题。
12

AI自动化官方学习入口

+

学习顺序：先飞书，再学一个普通AI工具和剪映；之后在扣子与Dify中选一个；最后才学n8n。

合规、隐私与发布前检查

+

宣传用语

不承诺“保证提分、保证名次、名师押题、包过”等结果，不虚构师资、荣誉和案例。

未成年人隐私

学生正脸、声音、姓名、学校、班级、成绩单编号等信息公开前必须获得明确授权并脱敏。

资质与平台规则

文化课培训的课程、时段、收费和招生宣传必须符合机构实际资质、平台类目和上海当地监管要求。

每次发布前五项检查

  1. 事实是否真实，课程、价格和名额是否准确。
  2. 学生和家长信息是否已授权并完成脱敏。
  3. 是否出现保证结果、夸大效果或无法证明的称号。
  4. 图片、音乐和评价是否具有使用权限。
  5. 行动入口是否清楚，平台是否允许该类内容或产品。
正式执行前：由机构负责人核实办学资质、课程安排、收费、合同与宣传口径。本方案提供运营方法，不替代法律、教育监管或平台合规意见。
